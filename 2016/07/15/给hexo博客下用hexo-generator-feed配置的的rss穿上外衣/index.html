给hexo博客下用hexo-generator-feed配置的rss添加样式

发表于 2016-07-15   |   分类于 nodejs   |  

问题

chrome访问rss,会提醒: 

This XML file does not appear to have any style information associated with it. The document tree is shown below.

本文主要解决rss没有样式的问题。

添加hexo-generator-feed插件

这里不详细说了,提一下注意事项:

  • npm feed插件的时候记得加上 –save
  • 根目录下的配置文件长这样:
plugin:
- hexo-generator-feed

#Feed Atom
feed:
    type: rss2  #或者atom
    path: rss2.xml #跟上面一致atom.xml
    limit: 20    #rss条目数最大值
    hub:   #这个可以不填
  • 配置完成,直接hexo g,会在public文件夹下生成xml文件

问题来了

当你打开自己网站,点击rss订阅有木有发现,会提示没有样式文件,然后列出了tree
所以重点了,本文叫你怎么添加样式文件,甚至在deploy -g的时候也是全自动生成的

解决问题

  • 在博客根目录/node_modules/hexo-generator-feed/ 文件夹下添加css/rss.css
  • 修改博客根目录/node_modules/hexo-generator-feed/lib/generator.js的代码
